Esctructura del Sitio

La estructura de esta SPA la realice con tecnología HTML5, CSS y JavaScript puro sin ningún tipo de librería adicional, esto con la finalidad de poder emplear todos los conocimientos adquiridos y estructuras bases que actualmente librerías como React emplea en la construcción de una SPA, pudiendo tener en cuenta cómo funcionan tras bambalinas estas librerías dividen el código en componentes y enrutadores para las diferentes paginas o consultas a cualquier API. Este sitio se estructuro tomando referencias como vanilla JS, la cual es una iniciativa que intenta enseñar las estructuras en código puro js y potenciando nuestras aplicaciones sin necesidad de añadir librerías extras, optimizando y mejorando la velocidad de carga del sitio web. Todos los códigos HTML, CSS y JavaScript los he aplicado totalmente desde cero los cuales he adquirido durante los últimos 4 años de estudios constantes en diferentes cursos, lecturas de manera autodidacta y teniendo en cuenta las constantes actualizaciones del Frontend.

A continuación, un ejemplo de las estructuras de carpetas del sitio, mostrada en la imagen:
App en el cual se encuentra el directorio Raiz, Assets como carpetas de (imágenes, recursos HTML, códigos CSS), componentes donde podemos encontrar todo lo relacionados a estructuras dinámicas y router para la construcción del sitio.

  • App: Directorio Raiz
  • Assets: Carpetas de imágenes, recursos HTML, códigos CSS.
  • Componentes: encontrando todo lo relacionados a estructuras dinámicas y router para la construcción del sitio.

Todas las consultas del sitio están basadas en la programacion del router.js tomando en cuenta el cambio de la URL (portafolio#/) con la finalidad de direccionar la consulta a la sección solicitada mediante consultas Ajax a los diferentes contenidos y API empleados.
Con esta breve explicación busco dar a conocer los conocimientos impartidos en las areas mencionadas.

Estructura CSS

En el contenido CSS se empleo maquetación Grid-CSS en la estructura principal del sitio, adicionando FlexBox y @media screen los cuales nos nos permite tener un mejor responsive en los diferentes resoluciones para todo tipo de dispositivos.
