Esctructura del Sitio

La estructura de esta SPA la realice con tecnología HTML5, CSS y JavaScript puro sin ningún tipo de librería adicional, esto con la finalidad de poder emplear todos los conocimientos adquiridos y estructuras bases que actualmente librerías como React emplea en la construcción de una SPA, pudiendo tener en cuenta cómo funcionan tras bambalinas estas librerías dividen el código en componentes y enrutadores para las diferentes paginas o consultas a cualquier API.
